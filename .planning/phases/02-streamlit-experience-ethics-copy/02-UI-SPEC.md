---

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Analyze Text |
| Empty state heading | Enter text to begin analysis |
| Empty state body | Paste at least 10 characters to run detector insights across the selected mode. |
| Error state | Analysis could not complete. Check input length and model readiness, then try again. |
| Destructive confirmation | Clear Input: This removes current text from the analysis box. Continue? |

Required limitations copy shown in every app sidebar:
- Results are probabilistic indicators, not definitive proof of authorship.
- Tool behavior is English-first and may be less reliable for other languages.
- Do not use this result as sole evidence in academic or legal decisions.

Required result-level reminder shown near verdict:
- Interpret this score alongside context, writing history, and human review.

---

## Interaction and Layout Contract

1. Each entrypoint follows the same section order:
   - Mode header
   - Input + Analyze action
   - Verdict/confidence/timing summary
   - Metrics and visualizations
   - Limitations reminder/footer
2. Analyzer-specific sections are allowed after shared summary blocks.
3. Result metrics must map directly to AnalysisResult contract fields.
4. Existing ChartGenerator methods are the only chart source for this phase.
